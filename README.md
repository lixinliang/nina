## Usage

* Pick one `*.xlsx` file.
* Pick one `*.pptx` file.
* Pick an output folder.
* Generate it.

## Rule

* The output `*.pptx` file is cloned from the template `*.pptx` file.
* In the template file, any text match `[A1]` would replace to correct text in the `A1` cell of `*.xlsx` file.
